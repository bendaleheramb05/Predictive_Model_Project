# Predictive Modeling Using Machine Learning

## Project Overview

This project uses Machine Learning algorithms to predict whether a passenger survived the Titanic disaster. The project demonstrates data preprocessing, model training, testing, and evaluation using supervised learning techniques.

---

## Objective

The objective of this project is to build a predictive model using Machine Learning algorithms and compare their performance.

---

## Dataset

- Dataset Name: Titanic Dataset
- Source: Kaggle
- Target Variable: Survived

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- VS Code

---

## Machine Learning Algorithms

- Decision Tree Classifier
- Random Forest Classifier

---

## Project Workflow

1. Import Required Libraries
2. Load Dataset
3. Data Cleaning
4. Handle Missing Values
5. Encode Categorical Data
6. Select Features and Target
7. Split Dataset into Train and Test
8. Train Decision Tree Model
9. Train Random Forest Model
10. Evaluate Model Performance
11. Visualize Results

2. Install required libraries

```bash

```

3. Run the project

```bash
python app.py
```
